Название проекта: Квантильная регрессия для цены автомобилей

Данные: 2500 автомобилей, цена в USD. Источник: датасет с GitHub.

Инструменты: R, пакеты tidyverse (обработка данных), quantreg (квантильная регрессия), ggplot2 (графики).

Результаты:

– Пробег почти не влияет на цену дешёвых автомобилей, но сильно снижает цену дорогих.
– Бренд (BMW, Tesla, Mercedes) добавляет наценку только в среднем и дорогом сегменте. Для дешёвых машин марка не важна.
– Старые премиальные автомобили стоят дороже молодых бюджетных, поэтому с возрастом цена не всегда падает.
– Объём двигателя важен для дешёвых машин, но не важен для дорогих.
